ESP32 API
Control API Firmware Upload

Use the /api endpoint with query parameters.

Streaming

  • /api?cmd=start – start video stream
  • /api?cmd=stop – stop video stream

Motor

  • /api?cmd=speed&val=0..255
  • /api?cmd=timeout&val=ms

Servos

  • /api?cmd=motor1&val=0..180
  • /api?cmd=motor2&val=0..180
  • /api?cmd=motor3&val=0..180

Light

  • /api?cmd=light&val=0..255

All commands return JSON.
